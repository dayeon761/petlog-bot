# Вет-бот (MVP)

Telegram-бот для владельцев кошек и собак в Иннополисе: напоминания о прививках/обработках
и проверка симптомов («ехать сейчас» / «сегодня-завтра к врачу» / «можно подождать»).

Бот — информационный сервис, не заменяет ветеринара.

## Запуск

```bash
pip install -r requirements.txt
cp .env.example .env
# впишите BOT_TOKEN (получить у @BotFather) в .env
python -m bot.main
```

## Важно перед реальным использованием

- В `bot/triage_data.py` список `CLINICS` — реальные круглосуточные клиники Казани (911,
  Зоосити, Ветбарс, ВетЭксперт), взятые из таблицы, предоставленной владельцем продукта.
  Периодически проверяйте, что адреса/телефоны не устарели.
- Даты прививок/обработок вводятся вручную (без фото/OCR) — так решили сознательно для
  первой версии MVP.
- Симптомы триажа основаны на общей ветеринарной практике неотложной помощи и присланном
  материале про признаки боли у кошек; это не заменяет консультацию с ветврачом при
  калибровке порогов.

## Структура

- `bot/config.py` — настройки (токен, интервалы, БД)
- `bot/db.py` — SQLite-схема и запросы (aiosqlite)
- `bot/states.py`, `bot/keyboards.py`, `bot/texts.py` — FSM, клавиатуры, тексты
- `bot/triage_data.py` — чек-листы красных флагов и признаков боли по видам (кошка/собака)
- `bot/handlers/start.py` — /start, /help, /cancel
- `bot/handlers/pets.py` — добавление питомца, список, отметки о прививках/обработках
- `bot/handlers/triage.py` — проверка симптомов
- `bot/reminders.py` — фоновый цикл напоминаний (прививки, обработки, чек-ап через 12ч)
- `bot/main.py` — точка входа
